# Excel to Tally Website 📊

Excel data को Tally में convert करने के लिए free tool की website।

## Files Structure

```
excel-to-tally/
├── index.html       ← Main website page
├── css/
│   └── style.css    ← Website design
├── js/
│   └── main.js      ← Website functions
└── README.md        ← Yeh file
```

## GitHub Pages पर Upload कैसे करें?

1. GitHub पर repository बनाओ: `yourusername.github.io`
2. सभी files upload करो
3. Settings → Pages → main branch select करो
4. Website live हो जाएगी!

## Customization

- `index.html` में अपना phone number और email change करो
- `css/style.css` में colors change कर सकते हो
- Download link को अपनी actual file के link से replace करो
